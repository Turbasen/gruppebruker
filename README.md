# Users and Groups CLI

CLI for creating new users and groups

## New User

Create new user for given user group

```
Usage: user [options]

Options:
   -g, --group     Group ID (Nasjonal Turbase)
   -n, --name      User name
   -e, --email     User email
   --ntb-api-env   API environment  [dev]
   --version       Print version and exit
```
